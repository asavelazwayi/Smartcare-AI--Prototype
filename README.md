# Smartcare-AI--Prototype
Modern healthcare AI prototype project created during AI Bootcamp
Smartcare AI Smartcare AI is a healthcare assistant designed to improve patient care using artificial intelligence.
It helps users with symptom guidance, appointment scheduling, and health monitoring in a simple and efficient way.

Features
AI symptom support
Appointment scheduling 
Patient health tracking
Medical report access 

Purpose 
To reduce waiting time, improve access to healthcare information, and support better patient care using AI technology. 

Future Improvements 
Mobile app version
Advanced AI diagnosis 
Doctor chat system
